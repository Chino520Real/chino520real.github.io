曼昆《经济学原理》

2025/10/25 Book Summary Economics

0. 引言

此文章为曼昆（N. Gregory Mankiw）《经济学原理》（Principles of Economics）第7版读书笔记。其中，这样子的词代表专业概念，我会用阅读时的知识架构用自己的话（或是书本原话，如果我觉得没有更好的表达方法时）试着解释，请不要当作教材学习喵🐱。

1. 经济学的十大原理

经济学概念

经济学 - 假设人都是贪婪的前提下如何分配有限的资源
稀缺性 - 资源一定是有限的，人类欲望一定是无限的，两者之间一定存在矛盾
效率 - 从稀缺资源中得到最大利益
平等 - 社会成员平分经济成果
机会成本 - 为了得到某些东西而必须放弃的东西
理性人 - 通过合理手段全力实现目标的人
边际变动 - 对计划的微小调整，可增可减
激励 - 某人做出某行为的原因

经济学十大原理

  1. 人们面临权衡取舍 - 想得到某东西，都必须放弃某东西
  2. 某种东西的成本是为了得到它所放弃的东西 - 所放弃的那个东西，就是机会成本
  3. 理性人考虑边际量 - 当边际收益大于边际成本时，理性人才会采取这个措施
  4. 人们会对激励做出反应 - 当人们看到实打实的收益或确凿的亏损（也就是激励）时，人们会做出反应
  5. 贸易可以使每个人的状况都变得更好 - 贸易使每个人（公司或国家）可以进行自身最擅长的活动，同时可以享有他人带来的更多物品与劳务
  6. 市场通常是组织经济活动的一种好方法 - 市场 = 企业决策 + 家庭决策，价格 + 个人利益 $\Rightarrow$ 合意
  7. 政府有时可以改善市场的结果 - 当市场本身无法自然调配资源分配情况时，政府的干预有助于市场恢复稳定
  8. 一国的生活水平取决于它生产物品与服务的能力 - 生产率 = 生产的物品数量（每小时） 与 服务数量（每小时） 的差别。生产率的增长率 决定了 平均收入的增长率
  9. 当政府发行了过多货币时，物价上涨 - 货币数量增加 => 货币价值下降 => 物价上涨
  10. 社会面临通货膨胀与失业之间的短期权衡取舍 - 通货膨胀率高时，失业率底；通货膨胀率低时，失业率高（菲利普斯曲线）

2. 像经济学家一样思考

为了更容易的研究，经济学家会假设世界上只有两个国家、或是只有两种产品，来更好的理解现实中复杂的情况。

循环流量图

循环流量图 - 简要说明货币如何通过市场在家庭和企业之间流动的宏观经济学模型

两类决策者：

  1. 家庭
  2. 企业

两种流向（两种循环）：

  1. 实物流 - 家庭提供生产要素 $\to$ 企业获得生产要素 $\to$ 企业生产产品 $\to$ 企业提供产品 $\to$ 家庭购买产品
  2. 货币流 - 家庭获得收入（工资） $\to$ 家庭用收入支付支出 $\to$ 企业获得收入（销售额） $\to$ 企业用收入支付报酬

两个市场：

  1. 物品与服务市场（家庭是买者、企业是卖者。企业提供工资、租金、利润等要素报酬）
  2. 生产要素市场（企业是买者、家庭是卖者。家庭提供劳动、土地、资本等生产要素）

生产可能性边界

生产可能性边界 - 假设可以得到的生产要素与技术均固定时，一个经济体所能生产的产品的产量的组合图

当经济体在生产可能性边界上生产时，代表有效的生产水平；当经济体在生产可能性边界以内生产时，代表没有充分利用所有可供利用的资源；生产可能性边界以外的区域经济体无法达到，因为没有足够的资源。同时，经济体不可能同时保证两种产品的量产。当技术进步时，生产可能性边界会往外扩张，代表了经济体所能生产的总量增大。

微观经济学与宏观经济学

微观经济学 - 研究对象为单个家庭以及单个企业（生产者）。研究内容为单个家庭如何利用有限的货币收入购买商品以获取最大满足，以及单个企业（生产者）如何利用有限的资源和劳务获取最大利益
宏观经济学 - 研究对象为国民经济总体。研究内容为经济宏观整体问题，如失业、通货膨胀、经济增长与波动、国际收支与汇率变动等

两者息息相关，出发点却不一样，所以采用不同的研究方法。

3. 相互依存性与贸易的好处

绝对优势 - 当一个生产者生产一种产品所需投入较少，则可以说该生产者在生产该产品上有绝对优势。劳动时间也算投入的一种，这时效率最高者有绝对优势。
比较优势 - 当一个生产者以低于另一个生产者的机会成本生产一种产品，则可以说该生产者在生产该产品上有比较优势
机会成本 - 假设某种资源在某用途上能获得收益，且该收益最大为X。为了将这个资源用于其他用途而放弃的收益X称为机会成本。

全球经济下，国家和个人之间会选择相互依存和贸易，是因为可以通过专业化生产和贸易解决由于稀缺性引起的问题。这里的专业化和贸易的好处是由比较优势带来的。

比较优势与机会成本：一种物品的机会成本是另一种物品的倒数，所以同一个人不可能在生产两个物品上同时拥有比较优势。
贸易价格：卖者可以以高于其机会成本的价格出售，买者可以以低于其机会成本的价格买入，双方均可获益。
各国之间的贸易遵循比较优势原理：美国生产粮食的机会成本低（土壤肥沃）而日本高，故美国应向日本出口粮食以换取日本进口的汽车。

4. 供给与需求的市场力量

市场 - （买者 + 卖者） who 交易 （物品 or 服务）

需求

需求量 - 消费者在各种可能的价格水平愿意且能购买的某商品数量
需求定理 - 物品价格上涨，需求量减少；物品价格下降，需求量增加
市场需求量 - 等于所有买者需求曲线水平相加的总和

影响需求曲线变动的因素：

  1. 收入
    • 正常物品 - 由收入增加引起需求量增加的物品
    • 低档物品 - 由收入增加引起需求量减少的物品
  2. 相关物品的价格
    • 替代品 - 假设A和B互为替代品。当A价格上升时，B需求量上升。替代品的例子：绿茶与红茶、苹果与香蕉
    • 互补品 - 假设A和B互为互补品。当A价格上升时，B需求量减少。互补品的例子：汽车与汽油、打印机与墨水
  3. 爱好。个人开始偏好另一种物品，那种物品的需求量增加
  4. 预期。对未来的收入、价格预期影响需求量
  5. 买者的数量。买者数量越多，需求量越大；买者数量越少，需求量越少

在价格既定时，买者希望购买的数量增多，需求曲线向右移动；买者希望购买的数量减少，需求曲线向左移动。如图所示，画一条$p=p_0$的直线代表价格既定，在这条线上，$q_1>q_0$

供给

供给量 - 生产者在各种可能的价格水平愿意且能出售的某商品数量
供给定理 - 物品价格上涨，供给量增加；物品价格下降，供给量减少
市场供给量 - 等于所有卖者供给曲线水平相加的总和

影响供给曲线变动的因素：

  1. 投入品价格（生产要素的价格）。投入品价格越高，供给量越低；投入品价格越低，供给量越高
  2. 技术。技术进步降低企业生产成本，进而增加供给量
  3. 预期。对未来的预期影响当期一种物品的供给量
  4. 卖者的数量。卖者数量越多，供给量越大；卖者数量越少，供给量越少

在价格既定时，卖者希望生产的数量增多，供给曲线向右移动；卖者希望生产的数量减少，需求曲线向左移动。如图所示，画一条$p=p_0$的直线代表价格既定，在这条线上，$q_1>q_0$

供给与需求的关系

供给曲线与需求曲线相交的点称作均衡状态，此时的价格称作均衡价格，此时的数量称作均衡数量

供求定理 - 任何物品的价格都会自发调整，使供需平衡。

  • 市场价格 > 均衡价格 => 物品过剩 => 市场价格下降
  • 市场价格 < 均衡价格 => 存在短缺 => 市场价格上升

均衡变动分析：

  1. 确定该事件是使需求曲线移动，还是供给曲线移动，还是两者都移动
    • 供给变动 - 供给曲线的移动
    • 需求变动 - 需求曲线的移动
    • 供给量的变动 - 沿着一条固定的供给曲线的移动
    • 需求量的变动 - 沿着一条固定的需求曲线的移动
  2. 确定往左还是右移动
  3. 将新均衡与原来的均衡做比较。新的均衡下价格和数量增加了还是减少了

在市场经济中，均衡价格是信号，决定了买者选择购买多少这种物品、卖者选择生产多少这种物品

5. 弹性与其应用

需求弹性

需求价格弹性 - 衡量需求量的相对变动对价格的相对变动的敏感程度（自变量：价格，因变量：需求量）
$$ 需求价格弹性 = \frac{需求量变动的百分比}{价格变动的百分比} $$
设变化前的均衡状态为$(Q_1, P_1)$，变化后的均衡状态为$(Q_2, P_2)$，则需求价格弹性使用中点法可以表示为
$$ 需求价格弹性 = \frac{\frac{Q_2 - Q_1}{(Q_2 + Q_1) / 2}}{\frac{P_2 - P_1}{(P_2 + P_1) / 2}} $$

影响需求价格弹性的因素：

  1. 是否有相近替代品 - 有相近替代品的物品更富有需求弹性；无相近替代品的物品更缺乏需求弹性
  2. 必需品与奢侈品 - 必需品倾向于缺乏需求弹性；奢侈品倾向于富有需求弹性
  3. 市场大小 - **小市场（如绿茶）的需求弹性大于大市场（如所有饮料）**的需求弹性
  4. 时间跨度 - 一般来说，物品在长期内更富有弹性

需求曲线与需求价格弹性的关系：
需求价格弹性与需求曲线的斜率密切相关。越陡峭，需求价格弹性越小；越平缓，需求价格弹性越大

  • (e) $需求价格弹性 = 0$，完全无弹性。不论价格如何变化都不影响需求量
  • (b) $需求价格弹性 < 1$，缺乏弹性。价格变化很难影响需求量变化
  • (c) $需求价格弹性 = 1$，单位弹性。价格变化率和需求量变化率一致
  • (a) $需求价格弹性 > 1$，富有弹性。价格变化会明显的影响需求量变化
  • (d) $需求价格弹性 = \infty $，完全富有弹性。价格低于均衡价格时，需求量无限；价格等于均衡价格时，需求量可以为任意数；价格大于均衡价格时，需求量为0

需求收入弹性 - 衡量需求量的相对变动对收入的相对变动的敏感程度（自变量：收入，因变量：需求量）
大于0的商品是正常品，收入越高，需求量越大；小于0的是低档品，收入越高，需求量越小
$$ 需求收入弹性 = \frac{需求量变动的百分比}{收入变动的百分比} $$

需求交叉价格弹性 - 衡量物品1的需求量的相对变动对物品2的价格的相对变动的敏感程度（自变量：物品2的价格，因变量：物品1的需求量）
大于0说明两种商品互为替代品（物品2价格增加 => 物品2需求量减少，但物品1需求量增加 => 人们开始用物品1代替物品2）；小于0说明两种商品互为互补品（物品2价格增加 => 物品2需求量减少，同时物品1需求量也减少 => 物品1需要配合物品2使用）；等于0说明两种商品没有关系
$$ 需求价格弹性 = \frac{物品1的需求量变动百分比}{物品2的价格变动百分比} $$

供给弹性

供给价格弹性 - 衡量供给量的相对变动对价格的相对变动的敏感程度（自变量：价格，因变量：供给量）
$$ 供给价格弹性 = \frac{供给量变动的百分比}{价格变动的百分比} $$

影响供给价格弹性的因素：

  1. 商品生产的灵活性 - 缺乏灵活性的供给缺乏弹性，如土地供给缺乏弹性
  2. 时间跨度 - 一般来说，物品在长期内更富有弹性，如新建工厂来增加产量

供给曲线与供给价格弹性的关系：
供给价格弹性与供给曲线的斜率密切相关。越陡峭，供给价格弹性越小；越平缓，供给价格弹性越大

  • (e) $供给价格弹性 = 0$，完全无弹性。不论价格如何变化都不影响供给量
  • (b) $供给价格弹性 < 1$，缺乏弹性。价格变化很难影响供给量变化
  • (c) $供给价格弹性 = 1$，单位弹性。价格变化率和供给变化率一致
  • (a) $供给价格弹性 > 1$，富有弹性。价格变化会明显的影响供给量变化
  • (d) $供给价格弹性 = \infty $，完全富有弹性。价格低于均衡价格时，供给量为0；价格等于均衡价格时，供给量可以为任意数；价格大于均衡价格时，供给量无限

弹性案例

  1. 谷贱伤农
    农业技术进步 $\Rightarrow$ 食物产量上升 $\Rightarrow$ 供给曲线向右移动（$S \to S^’$），但食物是必需品，缺乏需求价格弹性（$D$）$\Rightarrow$ 均衡价格大幅下跌（$P_1 \to P_2$），均衡需求量小幅增加（$Q_1 \to Q_2$）$\Rightarrow$ 总收益减少（$S_{Q_1Q_2E_2X} < S_{P_1P_2E_1X}$）
  1. 石油市场
  • 短期：石油输出国减少石油产量，且需求曲线陡峭（无石油替代品） $\Rightarrow$ 小额的数量减少引起大幅价格提升 $\Rightarrow$ 油价大幅上升
  • 长期：石油输出国减少石油产量，且需求曲线平缓（有石油替代品） $\Rightarrow$ 小额的数量减少引起的价格提升幅度变小 $\Rightarrow$ 油价回落
